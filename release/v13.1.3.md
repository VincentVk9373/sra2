# Release v13.1.3

**Shadowrun Anarchy 2 — FoundryVTT System**

Multi-target fixes for spells, removal of the obsolete "weapons-spells" feat type, and translation fix.

---

## Breaking Changes

### Removed "weapons-spells" feat type
The legacy "weapons-spells" feat type has been fully removed from the codebase. All references in character sheets, vehicle sheets, dice roller, and roll dialogs now only accept `weapon` or `spell`.

An automatic migration (13.1.3) deletes all remaining "weapons-spells" items from actors and from the world item collection.

---

## Bug Fixes

### Multi-target now works for direct spells
Direct spells (e.g. Déchirure Mentale) previously showed only a single defender in the chat message, even when multiple tokens were targeted. The "Apply Damage" button now appears for each targeted token.

### Fixed missing translation key
The orphan specialization hint on the character sheet displayed the raw key `SRA2.CHARACTER.NO_LINKED_SKILL` instead of the translated text. It now correctly uses `SRA2.SPECIALIZATIONS.NO_LINKED_SKILL`.

### Fixed range validation for spell rolls
The range validation check in the roll dialog had a typo (`weapons-spells` instead of `weapon-spell`) that could skip validation for certain item types. This is now cleaned up along with the type removal.
