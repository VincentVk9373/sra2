## AI Portrait Generation (Gemini)

### Setup
1. Get a Google Gemini API key (https://aistudio.google.com/apikey)
2. Go to **System Settings** â†’ **Gemini API Key** and paste the key
3. Open a character sheet in **advanced mode** (CTRL+E)
4. Click **"Generate AI Portrait"** below the avatar

### How It Works
- Generates **2 images**: a full body portrait (avatar) + a headshot (token)
- The token uses the portrait image as seed for visual consistency
- Images are saved in `assets/generated-portraits/`
- Works on **any actor** (generated runners, manual NPCs, creatures)
- The prompt is built from actor data (metatype, keywords, skills, equipment, armor, weapons, cyberware)
- Photorealistic rendering, Sixth World cyberpunk atmosphere

Video tutorial: https://youtu.be/xwdiBY4JHvA
