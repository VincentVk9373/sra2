# Release v13.3.1

## Bug fixes

### Validation error: `connectionMode: disconnected is not a valid choice`
- Added `disconnected` as a valid connection mode for characters
- Characters with a `disconnected` connection mode no longer cause a validation error on world load
- New `disconnected` mode appears in the connection mode selector on the character sheet (icon: `fa-plug-circle-xmark`)
- Biofeedback damage correctly uses matrix thresholds (Firewall) when disconnected, same as offline/AR
- Added EN/FR translations for the new mode
- Fixed FR translation: `OFFLINE` now says "Hors ligne" (previously "Déconnecté", which now maps to the new `DISCONNECTED` entry)
