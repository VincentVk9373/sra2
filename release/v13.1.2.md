# Release v13.1.2

**Shadowrun Anarchy 2 — FoundryVTT System**

Cleanup: remove the redundant and buggy "Vehicle / Drone" feat type.

---

## Breaking Changes

### Removed "Vehicle / Drone" feat type
The "vehicle" feat type has been retired. It was redundant with Vehicle/Drone actor types and had bugs. Vehicles and drones must now be managed exclusively through linked vehicle/drone actors (drag & drop a vehicle actor onto the character sheet).

An automatic migration (13.1.1) deletes all existing "vehicle" feat items when the world loads.
