# Release v13.1.0

**Shadowrun Anarchy 2 — FoundryVTT System**

This release brings a major new feature (Technomancer class), significant quality-of-life improvements, Dice So Nice integration, and numerous bug fixes based on community feedback.

---

## New Features

### Technomancer / Emerged Class
A fully integrated Technomancer class, mirroring the Awakened system:
- New **Emerged** feat type (level 7, 35,000 yen) with configurable abilities: Matrix Access + Virtual Persona, Complex Form Weaving, Sprite Compilation, Biofeedback
- New **Complex Form** feat type (5,000 yen each) — works like spells but uses **Technomancie + Logic**
- **Technodrain** — same mechanics as magical drain, triggered on Technomancie skill complications
- **Virtual Persona** — emerged characters get Firewall = Willpower, Attack = Charisma (used for matrix damage thresholds when no cyberdeck is equipped)
- **Biofeedback routing** — matrix damage is applied as physical damage for emerged characters
- **Essence sensitivity** — Technomancie tests are penalized by low essence (same thresholds as magic)
- **Compendium items** — Technomancie skill + 3 specializations (Formes complexes, Compilation, Decompilation)
- Full display in character sheet v2 with dice pool and RR calculations

### Dice So Nice Integration (#103, #104, #106)
- Custom Shadowrun-themed dice with success icons (5/6) and glitch icon (1)
- **Purple dice** = risk dice (with custom face icons)
- **Black dice** = normal dice
- Colors and labels applied directly per-roll to bypass DSN settings cache
- Dice animations respect Foundry roll mode (blind/self/whisper) — GM hidden rolls no longer show dice to players

### Essence on Character Sheet (#101)
- Essence value displayed below Charisma in the attributes section
- Value shown in the RR column in cyan, no dice icon
- Calculated automatically from `maxEssence - totalEssenceCost`

### Essence Penalties on Magic & Healing Rolls (#101)
- Magic tests: -1 success if essence <= 5, -1 more if essence <= 3
- Healing tests: -1 success if essence <= 4, -1 more if essence <= 2
- New `isMagic` checkbox on weapon feats for weapons used as spells

### Bio / Description Fields (#89)
- Bio and Notes free-text editors added to character sheet v2
- Visible only in advanced mode, using existing `bio.background` and `bio.notes` HTML fields
- Tabbed interface (Bio / Notes)

### Multi-Target Attacks (#19)
- Support for multi-target attacks with per-target defend and counter-attack buttons

### GM-Only Description (#43)
- GM-only description field on actors and items
- Visible only to the GM, not shared with players

### Cover Bonus (#88)
- Cover bonus field in defense roll dialog
- Added as defense successes (reduces damage), not extra dice
- Read on blur to avoid re-render while typing
- Hint text explaining the mechanic

### Spell Success Distribution (#79)
- Indirect spell success distribution between damage and zone effect

### Reference Footer (#80)
- Persistent reference footer on all item sheets

## Improvements

### Weapon Display (#83)
- VD and range stats shown under weapon names in character sheet
- Improved weapon stats layout with flex space-between

### Illusion Spell Subtypes (#55)
- Direct/Indirect renamed to Alteration/Psychotropic for illusion spells only

### Number Formatting (#107)
- All costs displayed with space thousands separator (390,000 -> 390 000 yen)
- New `{{formatNumber}}` Handlebars helper using fr-FR locale

### English Translation (#92)
- "Anarchy" renamed to "Edge" throughout the English translation

### Localization (#100)
- All hardcoded French strings in templates localized (vehicle context menu, specialization hints, weapon placeholders)

### Narration Actions (#102)
- Narration cost is now per-action: 1 action = +3, 2 actions = +6 to recommended level
- Migration 13.0.28 resets `grantsNarration` and compensates with +1 `narrationActions`
- Narration action count shown in cost breakdown

### RR Label (#8)
- Added RR label for clarity

### Chat Roll Actor
- When rolling from chat bar, uses player's assigned character or player name (no token fallback)

### Spell Weapons (#91)
- Better handling of weapons flagged as spells (`isSpell`, `isMagic` flags)

## Bug Fixes

- **#106**: GM rolls now respect Foundry roll mode — blind/self/whisper rolls hide both chat messages and DSN dice animations from players
- **#105**: Counter-attack fixes
- **#104**: Risk dice now correctly show purple color in Dice So Nice
- **#94**: Fixed Influence skill (VF) not found during RR creation
- Fixed `narrationActions` defaulting to 1 instead of 0, causing phantom +3 narration cost on all feats
- Fixed circular dependency crash when converting `character-sheet-v2` to static import
- Fixed DSN colorset not applying when user hadn't selected SRA2 system in DSN settings
- Improved damage flow handling

## Refactoring

- **Phase 1**: Extract magic numbers and hardcoded values into constants

- **Phase 3**: Split large functions in main system file
- **Phase 3.5**: Further code splitting
- **Phase 4**: TypeScript improvements
- Convert dynamic imports to static imports (except character-sheet-v2 due to circular dependency)
